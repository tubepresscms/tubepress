# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities **privately** through the contact form at **[tubepress.io/contact](https://tubepress.io/contact)** — mention "security" in your message so it is triaged first. Do not open public reports here (issues are disabled on this mirror in any case).

What helps us act fast:

- Affected version (see [tubepress.io/changelog](https://tubepress.io/changelog))
- Reproduction steps or a proof of concept
- Impact assessment as you see it

We aim to acknowledge reports quickly and to ship fixes in the next release. Security‑relevant changes are noted in the release notes.

## Supported versions

Only the **latest release** receives security fixes. The built‑in updater makes staying current a one‑click operation.
